# AI Academy Glossary

## A
**Agent** - An LLM combined with tools and goal-directed behavior, capable of taking actions.

**API** - Application Programming Interface; how software components communicate.

## C
**Chain-of-Thought** - Prompting technique where the LLM shows reasoning steps.

**Chunking** - Breaking documents into smaller pieces for processing.

**Context Window** - Maximum text an LLM can process at once.

## E
**Embedding** - Numerical representation of text for similarity comparison.

## F
**Fine-tuning** - Adapting a pre-trained model on specific data.

**Function Calling** - LLM's ability to output structured calls to external tools.

## G
**Grounding** - Connecting LLM to external data to improve accuracy.

**Guardrails** - Rules and filters that constrain AI outputs.

## H
**Hallucination** - When AI generates false but confident-sounding information.

## K
**KAF** - Kyndryl Agentic Framework; Kyndryl's AI platform.

## L
**LLM** - Large Language Model; AI trained on text to predict next words.

**LLMOps** - Operations practices for LLM-based systems.

## M
**MCP** - Model Context Protocol; standard for agent communication.

**Multi-agent** - System with multiple AI agents collaborating.

## O
**Orchestration** - Central controller directing multiple agents.

## P
**Prompt** - Input text given to an LLM.

**Prompt Engineering** - Craft of writing effective prompts.

## R
**RAG** - Retrieval-Augmented Generation; connecting LLM to external data.

**ReAct** - Reasoning + Acting pattern for agents.

## S
**System Prompt** - Hidden instructions shaping LLM behavior.

**Streaming** - Receiving LLM output token-by-token.

## T
**Temperature** - Parameter controlling output randomness (0=deterministic).

**Token** - Unit of text (~4 characters or 0.75 words).

**Tool** - External function an agent can call.

## V
**Vector Database** - Database optimized for embedding storage and search.

## Z
**Zero-shot** - Prompting without examples.
