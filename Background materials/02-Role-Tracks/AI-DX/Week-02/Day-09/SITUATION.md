# Day 9: Design System

## AI-DX Track - Week 2, Day 9

> Design & UX skill track supporting FDE, AI-FE and AI-PM (not a separate Designer job role in the Consult model).

### The Situation: "Every Screen Looks Different"

Multiple designers working on AI features.
Each screen has different buttons, colors, spacing.

**Challenge:** Create AI-specific design system components.

### AI Design System Components
- Chat bubbles (user/AI)
- Input patterns
- Loading states
- Error states
- Confidence indicators
- Source citations
- Feedback mechanisms

### Deliverables
1. Component specifications
2. Figma component library
3. Usage guidelines
