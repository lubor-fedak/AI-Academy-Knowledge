# Day 5: Map the Journey

## AI-DX Track - Week 1, Day 5

> Design & UX skill track supporting FDE, AI-FE and AI-PM (not a separate Designer job role in the Consult model).

### The Situation: "We Need a Journey Map"

Stakeholders keep arguing about features.
Nobody has documented what the user actually experiences.

**Challenge:** Create a journey map for an AI assistant interaction.

### Journey Map Elements
- Stages (before, during, after)
- User actions
- Touchpoints
- Emotions (highs and lows)
- Pain points
- Opportunities

### Week 1 Checkpoint
- User personas (3)
- Journey map (complete)
- Key insights summary
- Opportunity prioritization
