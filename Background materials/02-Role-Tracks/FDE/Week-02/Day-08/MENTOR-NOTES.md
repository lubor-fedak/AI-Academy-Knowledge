# Mentor Notes - FDE Day-08

## Key Outcomes
Students should be able to:
- Debug production issues systematically
- Deploy containerized AI applications
- Set up monitoring and alerting

## Watch For
- Skipping proper debugging methodology
- Not considering production constraints
- Ignoring security in deployment

## Intervention Timing
- Help if stuck >15 min on environment issues
- Let them struggle with design decisions
- Guide toward documentation when needed
