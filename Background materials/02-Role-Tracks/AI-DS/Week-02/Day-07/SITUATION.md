# Day 7: Detect the Bias

## AI-DS Track - Week 2, Day 7

### The Situation: "Is Our AI Fair?"

Legal asks: "Does our AI treat all users fairly?"
You need to check if performance varies across user groups.

**Challenge:** Conduct bias analysis on AI system.

### Bias Analysis Framework
1. Define protected attributes (gender, age, etc.)
2. Segment test data by attributes
3. Measure quality metrics per segment
4. Identify performance gaps
5. Investigate root causes

### Deliverables
1. Bias analysis methodology
2. Segmented performance report
3. Identified disparities (if any)
4. Mitigation recommendations
