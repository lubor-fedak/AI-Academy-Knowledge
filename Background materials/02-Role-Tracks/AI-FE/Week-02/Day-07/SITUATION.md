# Day 7: Make It Accessible

## AI-FE Track - Week 2, Day 7

### The Situation: "Our Disabled Employees Can't Use It"

HR reports: Employees using screen readers can't use the AI assistant.
The tool is unusable for anyone with visual impairments.

**Challenge:** Make the AI interface WCAG 2.2 compliant.

### Accessibility Requirements
- Screen reader compatibility
- Keyboard navigation
- Color contrast (4.5:1 minimum)
- Focus management
- ARIA labels for dynamic content

### Deliverables
1. Accessibility audit of current UI
2. Fixed implementation
3. Screen reader testing video
