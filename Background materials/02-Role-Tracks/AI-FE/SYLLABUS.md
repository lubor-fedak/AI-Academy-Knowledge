# AI Front-End Developer (AI-FE) Track Syllabus

## Role Overview

The AI Front-End Developer creates **AI-native user interfaces** that handle streaming, uncertainty, and real-time interactions gracefully.

This track describes a specialization within the Forward Deployed Engineer capability in the Consult role model.

### Core Responsibilities
- Build responsive, accessible AI interfaces
- Handle streaming responses and loading states
- Design for AI uncertainty (confidence, fallbacks)
- Optimize performance for AI-heavy applications
