## Glossary

| Term | Definition |
|------|------------|
| **Agent** | LLM + tools + goal-directed behavior |
| **Tool** | Function an agent can call (API, database, etc.) |
| **ReAct** | Reasoning + Acting pattern |
| **Function Calling** | LLM's ability to output structured tool calls |
| **Orchestration** | Central controller managing agents |
| **Choreography** | Agents coordinating peer-to-peer |
| **Planning** | Agent breaking down complex tasks |
| **Memory** | Agent's ability to remember past interactions |
