# Assessment Rubrics

## Artifact Quality Rubric

| Level | Score | Description |
|-------|-------|-------------|
| Excellent | 90-100 | Works correctly, well-structured, documented, handles edge cases |
| Good | 70-89 | Works correctly, reasonable structure, basic docs |
| Developing | 50-69 | Partially works, structure issues, minimal docs |
| Needs Work | <50 | Doesn't work, no structure, no docs |

## AI Conversation Quality Rubric

| Level | Score | Description |
|-------|-------|-------------|
| Excellent | 90-100 | Precise questions, iteration, challenges AI, explores alternatives |
| Good | 70-89 | Clear questions, uses responses, some exploration |
| Developing | 50-69 | Vague questions, accepts first response, limited exploration |
| Needs Work | <50 | Copy-paste, no iteration, no engagement |

## Reflection Quality Rubric

| Level | Score | Description |
|-------|-------|-------------|
| Excellent | 90-100 | Deep self-awareness, specific examples, growth mindset |
| Good | 70-89 | Shows learning, some specifics, positive attitude |
| Developing | 50-69 | Surface-level, generic statements |
| Needs Work | <50 | No reflection or irrelevant |

## Peer Review Quality Rubric

| Level | Score | Description |
|-------|-------|-------------|
| Excellent | 90-100 | Specific, actionable, balanced, constructive |
| Good | 70-89 | Mostly specific, some suggestions |
| Developing | 50-69 | Vague, one-sided |
| Needs Work | <50 | "Looks good" or no engagement |
