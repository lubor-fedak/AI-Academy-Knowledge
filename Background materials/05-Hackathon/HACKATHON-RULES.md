## Rules

### Team Composition
- 8 cross-functional teams
- 1 person per role in each team
- No team changes after kickoff

### Resources Allowed
- All academy materials
- AI Tutors (ChatGPT Enterprise)
- Antigravity IDE
- Azure resources (your team's resource group)
- Internet for documentation only

### Resources NOT Allowed
- External help (no friends, no contractors)
- Pre-built solutions from outside academy
- Sharing code between teams

### Deliverables
1. **Working Demo** (required)
2. **Architecture Diagram** (1 page)
3. **Value Proposition** (2-3 slides)
4. **Security Checklist** (completed)
5. **10-minute Presentation**

### Presentation Format
- 7 minutes: Demo + explanation
- 3 minutes: Q&A from judges

## Code of Conduct
- Respect all participants
- No sabotaging other teams
- Ask for help if stuck
- Have fun!
