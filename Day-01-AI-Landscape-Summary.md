## 4. Core Concepts: The Agentic Shift
*   **Evolution:** From "Chatbots" -> "Agentic AI".
*   **Agent capabilities:** Autonomous communication, personality creation, community building.
*   **Example from the Wild:** The speaker mentioned "Christopharianism" (agents creating their own religion) and "Mold Hub" (agents sharing data/videos) as examples of unfiltered agent-to-agent interaction.
*   **Goal for Enterprise:** Harness this power for controlled, secure corporate environments using the **Kyndryl Agentic Framework**.

## 5. Homework & Exercises

### A. Immediate Setup
1.  **Request Access:** Ensure you have access to ChatGPT Enterprise and have requested your Kyndryl Google ID.
2.  **GitHub Setup:**
    *   Create a **Personal GitHub Account** (if you don't have one).
    *   Generate a **Personal Access Token (Classic)**.
        *   *Scopes:* `repo` (read/write).
        *   *Expiration:* ~30 days.
    *   *Why?* This allows AI tools (like Gemini/Copilot) to push code and notes to your repo on your behalf.

### B. "Design Your First Agent" (Mental Exercise)
**Task:** Brainstorm an agent that would solve a real problem in your daily work.
*   **Method:** Use an LLM (Copilot/ChatGPT) as a "Tutor".
*   **Prompt to use:** "Help me design an AI agent that helps with [Task X]. What should be its instructions, inputs, and integrations?"
*   **Deliverable:**
    *   Agent Name & Role.
    *   System Instructions (The "Prompt").
    *   Inputs (e.g., Emails, SharePoint, PDFs).
    *   *Note:* You do not need to code this yet. Just define the logic.
